# TwoSum1 — LeetCode Java Solution

## 📁 Folder Structure

Ensure your project directory looks like this before running:

```
YourProject/
└── LeetCode/
    └── TwoSum1.java
```

---

## 🚀 Terminal Commands

### Step 1 — Navigate to the Parent Folder of `LeetCode/`

```bash
cd /path/to/YourProject
```

### Step 2 — Compile

```bash
javac -d . LeetCode/TwoSum1.java
```

### Step 3 — Run

```bash
java LeetCode.TestTwoSum1
```

## 📝 Notes

- Always run commands from the **parent folder** of `LeetCode/`, not from inside it.
- The `-d .` flag tells the compiler to respect the `package` structure.
- Use `java LeetCode.TestTwoSum1` (fully qualified class name) because the file has `package LeetCode;` declared.

---

## 🔧 Prerequisites

Make sure Java is installed:

```bash
java -version
javac -version
```

> Both should return a version number (e.g., `java 17.0.x`). If not, install the [JDK from Oracle](https://www.oracle.com/java/technologies/downloads/).
